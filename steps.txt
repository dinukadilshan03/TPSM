# 🔵 PHASE 1 — Understand the Business / Analytical Statement

Before touching the data:

### 1. Clearly Define the Analytical Statement

Example formats:

* “Discount percentage significantly affects sales quantity.”
* “Customer segment influences average order value.”
* “Sales can be predicted using pricing and discount variables.”

You must:

* Identify dependent variable (Y)
* Identify independent variables (X)
* Decide if it is:

  * Mean comparison problem
  * Relationship problem
  * Prediction problem
  * Time series problem

Without this clarity, analysis will be directionless.

---

# 🔵 PHASE 3 — Descriptive Analytics (Mandatory)

This answers: “What is happening in the data?”

You must compute:

### 1. Summary Statistics

For numerical variables:

* Mean
* Median
* Variance
* Standard deviation
* Skewness

Explain distribution shape.

### 2. Distribution Analysis (LO1)

Check:

* Is it Normal?
* Right-skewed?
* Left-skewed?

You may:

* Plot histogram
* Use normality tests (Shapiro, KS test)

### 3. Group Comparisons (Descriptive Only)

Example:

* Average sales by segment
* Average quantity by region

No hypothesis testing yet — just describe patterns.

---

# 🔵 PHASE 4 — Inferential Analytics (LO2, LO3, LO4)

Now you statistically validate the statement.

This is where many groups lose marks.

---

## Step 1: Formulate Hypotheses

Example:

H0: Discount has no effect on quantity
H1: Discount significantly affects quantity

You must:

* Define null hypothesis clearly
* Define alternative hypothesis clearly
* Set significance level (usually 0.05)

---

## Step 2: Choose Correct Test

Based on your variable types:

| Scenario                             | Test        |
| ------------------------------------ | ----------- |
| Compare 2 means                      | t-test      |
| Compare >2 means                     | ANOVA       |
| Relationship (continuous-continuous) | Correlation |
| Categorical association              | Chi-square  |
| Impact analysis                      | Regression  |

You must justify why the test is valid:

* Assumptions checked?

  * Normality
  * Independence
  * Equal variance

---

## Step 3: Regression Modelling (LO3)

If your statement involves impact or prediction:

### Linear Regression

Model form:
Y = β0 + β1X1 + β2X2 + ε

Interpret:

* Coefficient meaning
* p-values
* R²
* Adjusted R²
* F-test

Explain statistically:

* Is model significant?
* Is predictor significant?

Check assumptions:

* Linearity
* Homoscedasticity
* Normal residuals
* No multicollinearity

# 🔵 PHASE 6 — Interpretation & Justification (LO2, LO6 – Evaluate Level)

This is critical.

  * Sample size
